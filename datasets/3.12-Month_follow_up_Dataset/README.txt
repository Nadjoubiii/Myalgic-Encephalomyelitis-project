The following anonymised data files are provided:

"CFS_in_the_NHS 12_month_follow_up DATA_anonymised.csv" - contains baseline and 12-month follow-up data for N=952 patients
"CFS_in_the_NHS 12_month_follow_up VARIABLES_anonymised.csv" - variable definitions for "CFS_in_the_NHS 12_month_follow_up DATA_anonymised"

"CFS_in_the_NHS long_term_follow_up DATA_anonymised.csv" - contains baseline, 12-month and long-term follow-up data (if available) for N=1265 patients
"CFS_in_the_NHS long_term_follow_up VARIABLES_anonymised.csv" - variable definitions for "CFS_in_the_NHS long_term_follow_up DATA_anonymised"
